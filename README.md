# 量子五目並べ (Quantum Gomoku)

五目並べに「確率」と「観測」の概念を取り入れた、新しい戦略ボードゲームです。
石を置いた時点では色は確定しておらず、「観測」を行うことで初めて色が決定します。

## 🎮 ルール概要

### 基本ルール
*   通常の五目並べと同様、縦・横・斜めのいずれかに自分の色の石を**5つ**並べると勝利です。
*   ただし、盤上の石は**量子状態（確率的）**であり、見た目上の色が確定していません。

### 石の種類（確率）
プレイヤーはターンごとに以下の確率を持つ石を選択して配置します。

*   **先攻（黒）**
    *   **90%**: 90%の確率で黒になる（強い石）。**連続で使用することはできません。**
    *   **70%**: 70%の確率で黒になる。
*   **後攻（白）**
    *   **30%**: 30%の確率で黒になる（70%の確率で白）。
    *   **10%**: 10%の確率で黒になる（90%の確率で白、強い石）。**連続で使用することはできません。**

### 観測 (Observation)
*   プレイヤーは石を置いた後、任意で**「観測」**を行うことができます。
*   観測を行うと、盤面上の全ての石の色が確率に従って確定（収束）します。
*   **勝敗判定**:
    *   観測の結果、どちらかが5つ並んでいればそのプレイヤーの勝利です。
    *   もし両者が同時に5つ並んでいた場合、**観測を行ったプレイヤー（権利を行使した側）の勝利**となります。
    *   勝敗がつかなかった場合、盤面は観測前の量子状態に戻ります。
*   **回数制限**: 観測は1ゲームにつき、各プレイヤー**5回**まで可能です。

## 🕹️ 機能・操作方法

*   **PvP / PvEモード**: 
    *   ローカルでの対人戦（PvP）
    *   CPU対戦（PvE）に対応。CPUの先攻・後攻を選択可能。
*   **置き間違い防止モード**: タップで選択、もう一度タップで確定するモードへの切り替えが可能。
*   **待った機能**: 「待った」ボタンで手数を戻すことができます（CPU対戦時はプレイヤーの手番まで戻ります）。
*   **レスポンシブ対応**: PCおよびスマートフォンでのプレイに対応。

## 🚀 開発・実行方法

このプロジェクトは React + TypeScript + Vite で構築されています。

### インストール

```bash
npm install
```

### ローカルサーバー起動

```bash
npm run dev
```
ブラウザで表示されたURL（通常は `http://localhost:5173`）にアクセスしてください。

### ビルド

```bash
npm run build
```
`dist` フォルダに静的ファイルが生成されます。
